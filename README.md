This project only uses HTML, CSS(sass) and vanila Javascript.

## Usage

Make sure to have at lease Node v13.0+

### Install Dependencies

```
npm install
```

### Run

```
# Development mode (run at port 8080)
npm run start

# Production
npm run build
```
